About Me

I’m Khiry Dixon Manning, a software developer currently working at Apple, with a strong foundation from Atlas School and hands-on experience building full-stack applications and machine learning prototypes.

At Atlas School, I delved into both supervised and unsupervised learning techniques—implementing classification and clustering workflows—and applied these concepts in practical projects. My passion for creating interactive experiences led me to develop a text-based adventure website, where I designed game logic, user interaction flows, and a responsive UI (view project). I also built several Python-based games to reinforce algorithmic thinking and object-oriented design (see portfolio).

More recently, I engineered an Airbnb clone (v4) that mimics core platform features—listing management, search filters, user authentication, and data persistence—to deepen my understanding of RESTful design and scalable architectures (Airbnb clone).

At Apple, I collaborate with cross-functional teams to deliver reliable, high-quality solutions at scale. This role has sharpened my skills in code quality, testing, and performance optimization, as well as in agile workflows and peer reviews. I’m adept at translating product requirements into clean, maintainable code and enjoy mentoring peers through knowledge-sharing sessions.

My goal is to transition into a Machine Learning Engineer role, where I can combine my software engineering experience with data-driven model development. I’ve experimented with supervised learning (e.g., classification pipelines) and unsupervised approaches (e.g., clustering for insights) in personal and Atlas School projects, and I’m eager to work on end-to-end ML systems—data ingestion, feature engineering, model training, evaluation, and deployment.
